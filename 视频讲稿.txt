## **【演示操作脚本】（供演示人员参考）**

```
# TLC模拟器演示步骤
1. 导航至「反应监控」→「TLC模拟器」
2. 选择展开剂体系：PE/EA = 10:1
3. 点击"展开"按钮
4. 观察Rf值变化
5. 切换到"Vanillin显色"
6. 展示AI助手问答

# 柱色谱模拟器演示步骤
1. 导航至「分离纯化」→「柱色谱模拟器」
2. 点击"1. Pack Column"
3. 点击"2. Load Sample"
4. 调整溶剂极性滑块至50%
5. 点击"3. Start Elution"
6. 观察粒子分离动画
7. 展示馏分收集过程
```
